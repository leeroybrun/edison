# edison-qa-audit

Workflow: audit the QA record to see current state, rounds, and outstanding blockers.


```bash
edison qa audit $1
```


## Arguments

- **task_id** (required): Task identifier



## When to use

- You want to see what is blocking promotion
- You want to confirm whether validators agree
